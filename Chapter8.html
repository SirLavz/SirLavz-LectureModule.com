üìò Key Concepts

Set Operators:

  • UNION: Combines results, removes duplicates
  • UNION ALL: Combines results, retains duplicates
  • INTERSECT: Returns common rows
  • MINUS: Returns rows in first query not in second

Advanced Joins:

  • NATURAL JOIN: Matches columns with same names
  • JOIN USING: Specifies common column
  • JOIN ON: Explicit join condition
  • OUTER JOIN: Includes unmatched rows (LEFT, RIGHT, FULL)

üîç Subqueries & Correlated Queries

Subquery: A query nested inside another query. Can be used in WHERE, FROM, or SELECT clauses.

Correlated Subquery: References a column from the outer query. Executes once per outer row.

SELECT CUS_NAME
FROM CUSTOMER
WHERE CUS_BALANCE > (
  SELECT AVG(CUS_BALANCE)
  FROM CUSTOMER
);

üßÆ SQL Functions

  • String Functions: CONCAT, LENGTH, UPPER, LOWER
  • Date Functions: NOW(), DATEADD(), DATEDIFF()
  • Numeric Functions: ROUND, ABS, POWER
  • Conversion Functions: CAST, CONVERT

üß© Views, Triggers, and Stored Procedures

  • View: Virtual table based on a query
  • Trigger: Executes automatically in response to events
  • Stored Procedure: Precompiled SQL block stored in DBMS
