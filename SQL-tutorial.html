1. Introduction to SQL

SQL (Structured Query Language) is used to interact with relational databases. Common systems include MySQL, PostgreSQL, and SQL Server.

Core Commands: SELECT, INSERT, UPDATE, DELETE, CREATE, DROP

2. Basic SQL Syntax

SELECT column1, column2 FROM table_name WHERE condition;

3. SQL Data Types

TypeDescription
INTWhole number
VARCHAR(n)String of length n
DATEDate format
BOOLEANTrue/False
DECIMALPrecise decimal values

4. DDL - Data Definition Language

CREATE TABLE Students (
  ID INT PRIMARY KEY,
  Name VARCHAR(100),
  Age INT
);

5. DML - Data Manipulation Language

INSERT INTO Students (ID, Name, Age) VALUES (1, 'Ronald', 20);
UPDATE Students SET Age = 21 WHERE ID = 1;
DELETE FROM Students WHERE ID = 1;

6. DQL - Data Query Language

SELECT * FROM Students WHERE Age > 18;

8. SQL Joins

Types: INNER JOIN, LEFT JOIN, RIGHT JOIN, FULL OUTER JOIN

SELECT A.Name, B.Grade
FROM Students A
INNER JOIN Grades B ON A.ID = B.StudentID;
