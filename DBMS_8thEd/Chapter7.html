üìò Key Concepts

SQL (Structured Query Language) is a declarative language used to interact with relational databases. It includes commands for data definition, manipulation, and control.

SQL Categories:

  • DDL ‚Äì Data Definition Language (CREATE, ALTER, DROP)
  • DML ‚Äì Data Manipulation Language (SELECT, INSERT, UPDATE, DELETE)
  • DCL ‚Äì Data Control Language (GRANT, REVOKE)
  • TCL ‚Äì Transaction Control Language (COMMIT, ROLLBACK)

üßÆ SQL Syntax Essentials

  • Commands are case-insensitive but typically written in uppercase
  • Statements end with a semicolon (;)
  • String and date values are enclosed in single quotes
  • NULL represents unknown or missing values

Example:

CREATE TABLE CUSTOMER (
  CUS_ID INT PRIMARY KEY,
  CUS_NAME VARCHAR(50) NOT NULL,
  CUS_PHONE CHAR(10),
  CUS_BALANCE DECIMAL(9,2) DEFAULT 0.00
);

üîê SQL Constraints

  • NOT NULL: Prevents null values
  • UNIQUE: Ensures all values are distinct
  • DEFAULT: Assigns a default value
  • CHECK: Validates data against a condition
  • PRIMARY KEY: Uniquely identifies each row
  • FOREIGN KEY: Enforces referential integrity
