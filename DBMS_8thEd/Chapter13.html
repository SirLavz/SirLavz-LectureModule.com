üìò Key Concepts

Business Intelligence (BI): A framework that transforms data into information, information into knowledge, and knowledge into strategic decisions. It includes tools for data capture, integration, analysis, and visualization.

Operational vs. Decision Support Data:

  • Operational: Current, detailed, transaction-based
  • Decision Support: Historical, summarized, multidimensional

üè¢ Data Warehouse Architecture

  • Subject-Oriented: Organized by business topics
  • Integrated: Consolidates data from multiple sources
  • Time-Variant: Stores historical data
  • Nonvolatile: Read-only for analysis

Data Mart: A smaller, focused subset of a data warehouse for specific departments or functions.

‚≠ê Star Schema

A relational model for multidimensional data analysis. It includes:

  • Fact Table: Central table with numeric measures
  • Dimension Tables: Describe business perspectives (e.g., time, product, location)
  • Attributes: Qualify dimensions
  • Hierarchies: Enable drill-down and roll-up operations

üîç OLAP & Data Mining

  • OLAP: Online Analytical Processing ‚Äì supports multidimensional analysis, modeling, and forecasting
  • ROLAP: Relational OLAP ‚Äì uses relational databases
  • MOLAP: Multidimensional OLAP ‚Äì uses cube structures
  • Data Mining: Discovers hidden patterns and relationships using statistical and AI techniques

üßÆ SQL Extensions for OLAP

  • GROUP BY CUBE: Generates all possible combinations of groupings
  • GROUP BY ROLLUP: Creates subtotals and grand totals
  • RANK(), DENSE_RANK(): Analytical ranking functions
  • LEAD(), LAG(): Access data from preceding or following rows
