üìò Key Concepts

Systems Development Life Cycle (SDLC): A framework for developing information systems, including planning, analysis, design, implementation, and maintenance.

Database Life Cycle (DBLC): A structured approach to database development with six phases:

  1. Initial Study
  2. Database Design
  3. Implementation and Loading
  4. Testing and Evaluation
  5. Operation
  6. Maintenance and Evolution

Design Strategies:

  • Top-Down: Starts with high-level data models and refines into details
  • Bottom-Up: Begins with detailed data elements and builds upward
  • Centralized vs. Decentralized: Depends on organizational structure and data distribution

üß© Design Phases

  • Conceptual Design: Abstract model using ER diagrams and normalization
  • Logical Design: Maps conceptual model to DBMS-specific structures
  • Physical Design: Defines storage, indexing, and performance parameters

Verification: Ensures model aligns with business rules and system processes
