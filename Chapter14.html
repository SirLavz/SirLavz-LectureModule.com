📘 Key Concepts

Database Connectivity: Mechanisms that allow applications to connect and interact with data sources. Common technologies include:

  • ODBC: Open Database Connectivity – standard API for accessing databases
  • OLE-DB: Object Linking and Embedding for databases – supports diverse data sources
  • ADO.NET: .NET framework for disconnected data access using XML
  • JDBC: Java Database Connectivity – Java API for relational databases

🔗 Web-to-Database Middleware

Middleware connects web servers to databases. Common types include:

  • CGI: Common Gateway Interface – slower, external scripts
  • API: Application Programming Interface – faster, integrated
  • Web Application Servers: Provide security, authentication, and dynamic content

🖥️ Client-Side Extensions

  • Plug-ins: External apps invoked by browser
  • Java & JavaScript: Embedded scripts activated by events
  • ActiveX & VBScript: Windows-oriented embedded scripts

📄 XML and Web Data Exchange

XML: Extensible Markup Language – defines custom tags for structured data exchange

  • DTD: Document Type Definition – defines valid structure and tags
  • XML Schema: Advanced structure definition using XSD
  • Presentation Separation: XML separates data from display logic

🧪 Laboratory Activity: Connecting Web Forms to a Database

💡 Objectives
  • Build a web form that connects to a database using middleware
  • Use server-side scripting to process form data
  • Store and retrieve data using SQL queries
📋 Instructions
  1. Create a simple HTML form with fields: Name, Email, Message
  2. Use PHP or Node.js to process form submissions
  3. Connect to a MySQL or SQLite database using ODBC or native drivers
  4. Insert submitted data into a table
  5. Display stored messages on a separate page
  6. Submit code and screenshots of working demo
