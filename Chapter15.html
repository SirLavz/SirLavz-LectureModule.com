ğŸ“˜ Key Concepts

Database Administration involves both managerial and technical functions to ensure data integrity, availability, and security.

  • Managerial Role: Planning, policy enforcement, resource allocation, end-user support
  • Technical Role: DBMS installation, configuration, maintenance, performance tuning
  • Security Framework: Confidentiality, integrity, availability
  • Security Vulnerabilities: Technical, managerial, cultural, procedural weaknesses

ğŸ“‘ Policies, Standards, and Procedures

  • Policies: Broad directives (e.g., password renewal every 6 months)
  • Standards: Minimum requirements (e.g., password length)
  • Procedures: Step-by-step instructions (e.g., account creation workflow)
