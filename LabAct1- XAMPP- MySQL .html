IS 311 – Information Management (DB/SQL)

Laboratory Activity 1.2: Familiarization of XAMPP MySQL UI

Learning Objectives
  • Identify the components and interface of XAMPP Control Panel and phpMyAdmin.
  • Start and configure Apache and MySQL services using XAMPP.
  • Navigate and explore the phpMyAdmin interface for managing MySQL databases.
  • Create, view, and delete a simple database and table using the UI.
  • Execute basic SQL queries using the SQL tab in phpMyAdmin.
Materials Needed
  • Laptop or desktop computer
  • Installed XAMPP software
  • Web browser (Chrome, Firefox, Edge)
  • Activity worksheet (provided by instructor)
Step-by-Step Procedures
  1. Open XAMPP Control Panel and start Apache & MySQL services.
  2. Open a browser and go to http://localhost/phpmyadmin.
  3. Explore phpMyAdmin interface: dashboard, tabs, database list.
  4. Create a new database named student_db.
  5. Create a table named students with 4 fields: id, first_name, last_name, course.
  6. Insert one record using the Insert tab.
  7. Browse the table to view the inserted record.
  8. Use the SQL tab to run: SELECT * FROM students;
