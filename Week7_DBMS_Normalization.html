What is Database Normalization?

Database Normalization is the process of organizing data in a relational database to:

  • Reduce redundancy
  • Improve data integrity
  • Prevent anomalies during INSERT, UPDATE, and DELETE operations

It was introduced by Edgar F. Codd and is applied in stages called Normal Forms (NF) ‚Äî each stage builds on the previous one.
