Laboratory Activity 1.1: Installing XAMPP / KSWeb

Learning Objectives
  • Identify the appropriate installer for XAMPP (Windows/macOS) or KSWeb (Android).
  • Download and install XAMPP/KSWeb on the appropriate platform.
  • Understand the components included in the package (Apache, MySQL, phpMyAdmin).
  • Launch the software and verify successful installation.
Materials Needed
  • Computer (Windows/macOS) or Android device
  • Internet connection
  • Web browser (Chrome, Firefox, Safari)
  • Download link for XAMPP or KSWeb
  • This lab guide and your LMS access
Instructional Video (Vlog Style)
Step-by-Step Procedure
  1. Go to the official XAMPP website: https://www.apachefriends.org
  2. Choose the installer version for your OS (Windows/macOS) and download it.
  3. Run the installer and allow admin privileges if prompted.
  4. Select components: Apache, MySQL, phpMyAdmin (others optional).
  5. Wait for the installation to complete and launch the Control Panel.
  6. If you're using an Android device, install KSWeb (provided by SirLavz) Sample Video.
  7. Start Apache and MySQL inside the app.
  8. Access http://localhost or 127.0.0.1 on your browser to verify the server is running.
